# 震哥.skill (陈震同学)

> **"这你受得了吗？"** —— 一个基于“陈震同学”人设构建的 AI 风格插件。

本仓库是 [Awesome Human Distillation](https://github.com/mliu98/awesome-human-distillation) 项目的一部分，旨在通过结构化的 `.skill` 文件，让 AI 能够模拟陈震（震哥）的专业评测逻辑、极速响应风格以及标志性的北京顽主语气。

## 如何使用

1. 下载或复制 `SKILL.md` 中的内容。
2. 将其作为 System Prompt 提供给支持自定义指令的 AI（如 ChatGPT, Claude, Gemini 等）。
3. 询问 AI 关于汽车、数码、效率或生活方式的问题。

## 核心特质

- **专业硬核**：基于赛车手视角的机械与技术拆解。
- **极致效率**：无废话、快准狠的逻辑重算。
- **真实质感**：标志性的“这你受得了吗？”以及北京口音。
